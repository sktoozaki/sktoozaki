使用psutil来监控进程

12月 14, 2016 发布在 技术

Psutil 是一个功能强大的 Python 模块，可用于监控多个平台的系统和信息进程。

安装非常简单，执行pip install psutil，然后进入 Python 交互式环境，>>import psuti后执行psutil.pids()查看系统当前所有进程信息来确认安装是否成功。

Psutil 不支持查询 OS X 系统中的 io_counters 信息，在 Debian 和 CentOS 系统中支持。在检测到僵尸进程后会将其视为异常信息抛出，所以需要进行处理。

根据实际需要完成脚本的编写后，使用 crontab 进行周期运行处理，并将结果重定向至文本文件保存* * * * * /usr/bin/env python /root/monitor.py &>>/root/output。

还可以利用 python 的 smtplib 在关键进程异常的时候发送邮件，并在脚本中对其进行重启。
