Mac 配置 Apache+MySQL+PHP

6月 04, 2016 发布在 技术

在 Windows 下开发的时候，不想捣腾配置 Apache + MySQL + PHP, 直接用的是国人开发的 phpStudy, 可以选择 Apache/NGINX, 还可以分别设置软件版本，非常方便，感觉比法国人做的 WampServer 好用。

在 OS X 下，也有类似如 MAMP 和 XAMPP. 不过 OS X El Captian自带了 Apache 和 PHP，方便了不少，索性安装 MySQL 来配置一番。

在 MySQL 官网下载 Mac OS X 10.11 DMG Archive，安装后直接可以在系统偏好设置中启动。
